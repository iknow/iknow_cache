# IknowCache

iKnow's versioned nested cache backed by Rails cache

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'iknow_cache'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install iknow_cache
```
